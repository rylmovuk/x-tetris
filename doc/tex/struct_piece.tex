\doxysubsection{Detailed Description}
Struct representing a piece \char`\"{}that we care about\char`\"{}, containing its shape and coordinates. Note that a valid state {\itshape can} contain x and/or y that are out of bounds relative to the game board\+: but in that case any cell at {\ttfamily shape\mbox{[}i\mbox{]}\mbox{[}j\mbox{]}} {\itshape must} be 0 if {\ttfamily board\mbox{[}y+i\mbox{]}\mbox{[}x+j\mbox{]}} is out of bounds.
