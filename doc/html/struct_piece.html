Detailed Description

Struct representing a piece "that we care about", containing its shape and coordinates. Note that a valid state can contain x and/or y that are out of bounds relative to the game board: but in that case any cell at shape[i][j] must be 0 if board[y+i][x+j] is out of bounds.
